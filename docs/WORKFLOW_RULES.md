# GraffiT Workflow Rules

Эти правила считаются базовыми для всех следующих итераций работы.

## Основной подход

- Работаем в репозитории: `https://github.com/Antongotry/GraffiT`.
- Архитектура темы повторяет подход `constalt`: page templates, `template-parts`, SCSS partials, один `main.js`.
- Повторяющиеся блоки выносим в reusable partial/component, а не дублируем по шаблонам.

## Правила размеров

- Desktop: перевод размеров в `vw` относительно макета `1440`.
- Mobile до `1024px`: перевод размеров в `vw` относительно макета `375`.
- Технические значения вроде `1px` бордеров, линий и деликатных штрихов можно оставлять в `px`.

## Текущий фокус

- Первая собираемая страница: `/services/`.
- Первый собранный блок: hero секция страницы услуг.
